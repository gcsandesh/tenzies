# Tenzies

Tenzies game that I created while learning react from Scrimba.

## How to play?

Roll until all dice are same. You need to touch a die to hold it and prevent from rolling. Once all dice are same and held, you win the game.

## To install packages

`npm install`

## To start the app

`npm run start`
